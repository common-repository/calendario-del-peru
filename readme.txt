Muestra eventos del día del Calendario peruano. Este plugin mostrará 5 titulares del Calendario, todos los días.

== Description ==

Muestra los eventos del día en el <a href="https://www.deperu.com/calendario/">Calendario peruano</a> publicados en DePeru.com

*El Calendario incluye los días internacionales, celebraciones nacionales, efemérides, etc.*

Este plugin mostrará 5 eventos del Calendario, todos los días. Es muy probable que algunos días tengamos registrados más de 5 eventos, pero solo se mostrará esa cantidad. Si observa menos es porque no tenemos registrados más eventos.

**Observaciones del Calendario:**

*   Todos los titulares tienen un link hacia el artículo correspondiente.
*   Muestra 5 eventos en forma predeterminada, a menos que tengamos menos notas registradas en el sitio.
*   Próxima versión mostrará si el día es feriado, permitirá aumentar o disminuir los eventos mostrados, permitirá cambiar el título, etc.
*   Funciona con todos los temas, usa los colores y tamaños del tema que se esté usando.

Muchas gracias por usar nuestro calendario, le recomendamos visite nuestra sección de <a href="https://www.deperu.com/noticias/">noticias de actualidad</a> en nuestro portal.

**CONFIANZA**
Este plugin *no recoge datos y no muestra publicidad* dentro del blog instalado. La privacidad de sus visitantes está garantizada.

== Installation ==

Instalar este plugin es sumamente sencillo, solo acepte instalar el plugin, luego coloque el widget en el lugar que desee dentro de la plantilla de su tema.

Pasos en detalle:

1. Instalar el plugin.
2. Ir al Menú de Apariencia y elegir Widgets.
3. Arrastrar el bloque de "Calendario del Perú" hacia la derecha donde están los sidebar y footer del diseño.

**PRIVACIDAD**
Este plugin no recoge datos y no muestra publicidad dentro del blog instalado.

== Frequently Asked Questions ==

= ¿Se puede personalizar el widget? =

Lo sentimos en esta versión no es posible hacerlo, prometemos tener opciones de personalización para las próximas versiones.

= ¿Tienen eventos para todos los días del año? =

¡Sí!, somos el único portal peruano en tener un calendario con contenido para los 365 días del año. Nuestros redactores en <a href="https://www.deperu.com">www.deperu.com</a> trabajan todos los días para agregar más eventos, incluso la información de los días son actualizados constantemente.

= ¿Qué otros temas podemos encontrar en DePeru.com? =

Nuestro portal tiene mucho contenido para Usted, desde noticias, calendario, calculadora del IGV-Alcabala, temas para las tareas de los estudiantes, contenido para pregrado, grado y postgrados universitarios, directorios de comercios, etc. Somos fuente  para muchas instituciones académicas. Proveemos tecnología a muchas Startups.
